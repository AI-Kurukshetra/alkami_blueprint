# NextGen Digital Banking Experience Platform PRD

## Vision
Build a production-grade SaaS digital banking platform for mid-size banks and credit unions that supports retail banking journeys, operational controls, and advanced financial services from a single Next.js and Supabase stack.

## Goals
- Deliver a deployable multi-app monorepo for customer and admin experiences.
- Support secure authentication, account views, transactions, transfers, bill payments, cards, loans, notifications, and support.
- Establish a banking-grade data model with auditability, RLS, and fraud/compliance hooks.
- Provide an expansion path for AI insights, open banking APIs, real-time payments, and automation.

## Mandatory Stack
- Next.js 14 App Router
- TypeScript
- Tailwind CSS
- shadcn/ui-style components with Radix UI primitives
- Supabase Postgres, Auth, Storage, Edge Functions, and RLS
- TanStack Query
- Zod validation
- Vitest and Playwright
- Vercel deployment
